# SystemMonitor

SystemMonitor è un monitor di sistema che permette di visualizzare informazioni dettagliate sull'utilizzo delle risorse del computer

---

## Obiettivo

L'obiettivo è creare un monitor di sistema più completo di quelli già presenti su Windows, pensato per chi vuole dati più precisi e più informazioni, mantenendo comunque delle buone prestazioni

In futuro, se il progetto mi piacerà e continuerò a lavorarci, vorrei fare anche una versione per Linux
