# SystemMonitor

SystemMonitor è un monitor di sistema che permette di visualizzare informazioni dettagliate sull'utilizzo delle risorse del computer su **Windows** e **Linux**

---

## Obiettivo

L'obiettivo è creare un monitor di sistema più completo di quelli già disponibili, pensato per chi vuole dati più precisi e più informazioni, mantenendo comunque delle buone prestazioni
